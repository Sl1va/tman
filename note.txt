Question: where to save user specific config file?
Answer:
    a) ~/.config - for user specific config files
    b) /etc      - "static" system-wide configs
    c) /var      - for data files generated during execution
Example from `man ssh_config`
    1.   command-line options
    2.   user's configuration file (~/.ssh/config)
    3.   system-wide configuration file (/etc/ssh/ssh_config)
